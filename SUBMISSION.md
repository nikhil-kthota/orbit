# Assignment Submission: Orbit Mission Control

Orbit is a full stack task management system built with a focus on clean design and secure data handling.

## Deliverables

1. **GitHub Repository**: This repository holds the complete source code for both the client and the server.
2. **Authentication Flow**: A working system for user registration and login using JWT and bcrypt for security.
3. **CRUD Dashboard**: A feature rich dashboard where users can manage missions (tasks) with full create, read, update, and delete support.
4. **API Documentation**: Detailed documentation for all backend routes can be found in [API.md](./API.md). I have also included a [Postman Collection](./Orbit_API_Collection.json) for easy testing.
5. **Scaling Strategy**: A specific plan on how to host and scale this integration for production can be found in [SCALING.md](./SCALING.md).

## Technical Overview
- **Frontend**: React, Vite, and Tailwind CSS.
- **Backend**: Node.js and Express.
- **Database**: Supabase (PostgreSQL) with Row Level Security.
- **Deployment**: Both services are hosted on Vercel and connected via environment variables.

Everything is currently live and the setup instructions are in the primary README.
